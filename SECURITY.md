# Security policy

This is experimental software deployed on a testnet. It has not been audited.
Do not use it with funds you are not prepared to lose.

## Reporting a vulnerability

Please report suspected vulnerabilities privately via GitHub Security Advisories
on this repository rather than opening a public issue. Include:

- a description of the issue and its impact,
- the affected component (contract, SDK, facilitator),
- a reproduction or proof of concept if available.

## Scope

- `EscrowPaymentGateway.sol` accounting, signature verification, and withdrawal
  cooldown logic.
- Voucher construction and verification in the SDK and facilitator.
